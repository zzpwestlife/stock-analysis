# 双侧美股突破策略（Youtube 双侧玩美股）
 
## 简介

我的突破策略是从 Qullamaggie 的 Break Out Strategy 中演变出来的，基本与他的策略相符，但根据我自己的一些观察增加了一些要求和策略细节，个人认为更适合大多数人 follow。我想要找的 Setup（突破前的准备形态）主要有以下几个部分构成

1. 一只股票需要在过去的一段时间现有一波大幅的上涨，上涨越线性越好，而且我希望他能够在 MA10/MA20 经常找到支撑。如果上涨中途经常插大阴线，则减分，因为股性相对来说就更差了。
2. 在上涨之后，该股票需要有一波中幅的回调，回踩 MA10/MA20（稍微大盘一些的 MA50 也可以）。之后，该股票需要在均线附近找到支撑并进行盘整，在盘整的尾部，K 线排列需要变得很紧，价格波动幅度需要缩小。 
3. 在从高点算至少 9-10 个交易日之后，且离 50 日均线的距离小于 35%，股票突破盘整趋势线，这个时候交易量并不是很关键的参考指标，当然，有放量更好。
4. 突破的时候直接买入，并把止损设在当天的低点，这个止损不应该很宽，你的 risk 也不应该大于整体账户的 1%，风险回报比十分重要，如果股票已经涨了很多了，应该放弃参与，管住手。
5. 在有盈利后的 3 个交易日后，卖出 50% 的仓位，剩余仓位以 MA10 做移动止盈，收盘价跌破 MA10 清仓离场，在触发之前，可以逢高略微减仓，但确保留有至少 1/3 的底仓等触发移动止盈再卖出。
 
## 如何找到这些股票？ 

在看盘软件里，你应该有一个自己的筛选条件，找到过去一个月最强势的股票。如果股票走势比较干净，放在观察列表里，等待 Setup 的形成。

这是我 FUTU 里面的筛选指标 

- 所属市场：美股
- 20 日涨跌幅（%）：>= 20.00 
- 振幅（%）近 20 日平均：>=6
- 1 日成交额（万）：>= 1000
- 成交额（万）- 近 20 日平均：>=1000

 
## 注意事项!!
 
交易突破时，必须确保纳指或罗素（因为大多数符合的股票都在这两个指数中）的 MA10 在 MA20 以上，并且两根均线都在上升趋势中。大盘越强，突破的成功率就越高。注意分仓，永远不要 ALL IN 一只股票，确保每笔交易的 Risk 在账户的 1% 及以下
